# Custom RC Car

A custom-built RC car engineering project designed from scratch using PVC, 3D-printed parts, steel shafts, gears, and basic electronics.

This project is not based on a commercial RC chassis. The goal is to design, build, test, and improve each subsystem step by step.

## Project Goals

- Build a functional custom RC car platform
- Design a printable 1:1 steering mechanism
- Use PVC and simple workshop tools where possible
- Learn practical mechanical design through real testing
- Document every module clearly

## Build Method

The project follows an iterative engineering method:

1. Learn the minimum theory needed
2. Design the module
3. Build it by hand
4. Test it
5. Improve the design

## Main Modules

- Steering
- Chassis
- Rear drivetrain
- Electronics
- Body

## Current Stage
